3. Provide an example of when the figure element would be useful in an HTML document.


The <figure> element is useful when you want to group media content (like an image, diagram, chart, code snippet, or illustration) together with a caption that describes it.
It keeps the content and its explanation semantically connected.
<figure>
  <img src="mountains.jpg" alt="Snow-covered mountains at sunrise">
  <figcaption>
    Sunrise over the Rocky Mountains during winter.
  </figcaption>
</figure>
